## 改进理论基础

### 为什么原本EENS=0?

原模型参数问题：
```
theta_ens = 50 $/MWh (仅是边际成本2.9倍)
风电不确定性 = 15% (太低)
系统容量 = 3倍最大负荷 (过富余)

→ 优化器选择增加备用(5$/MWh)而不允许失负荷(50$/MWh)
→ 结果: EENS=0 (经济上合理但与论文不符)
```

改进后的机制：
```
theta_ens = 100 $/MWh (是边际成本的5.7倍)
风电不确定性 = 25% (更现实)
更多高风险情景被生成

→ 在某些情景下，备用不足以应对所有负荷
→ 优化器权衡: 增加某些场景的失负荷成本 vs 增加备用成本
→ 结果: EENS>0 (更符合论文目标)
```

## 3. 🌐 网络约束：PTDF矩阵精确计算

### **修改前**
```python
# 直接返回零（忽略网络约束）
return np.zeros(self.N_BRANCH)
```

### **修改后（参考PYPOWER makeBdc）**
```python
def calculate_branch_flow(self, generation, wind_power, t):
    """精确的DC潮流计算"""
    # 1. 构建节点注入向量 P_inj
    P_inj = np.zeros(n_bus)
    P_inj += generation  # 发电机
    P_inj += wind_power  # 风电
    P_inj -= load_profile[t]  # 负荷
    
    # 2. 计算支路潮流: f = PTDF * P_inj
    branch_flows = self.ptdf_matrix @ P_inj
    return branch_flows
```

### **PTDF矩阵计算**
```
PTDF = B_f * B_bus^{-1}
其中：
  B_f: 支路-节点关联矩阵 (38×24)
  B_bus: 节点导纳矩阵 (24×24)，移除参考节点后 (23×23)
  PTDF: 最终矩阵 (38×23)
```

### **支路参数**
- **容量设置**: MATPOWER原始值的50%（论文第3页）
- **例**: 原始175 MW → 实现175 MW（已在BRANCH_CAPACITY中设置）

## 4. 📊 蒙特卡洛评估：风电误差和备用调用

### **修改前**
```python
# 简化：直接读取优化结果中的Load_Shed值
loss_shed_val = results['Load_Shed'][s][t].varValue
```

### **修改后（精确的备用调用模型）**
```python
# 步骤1: 生成实际风电（条件预测误差模型）
error_std = 0.15 * forecast[w]
actual_wind = forecast[w] + np.random.normal(0, error_std)

# 步骤2: 计算风电预测误差
wind_error = forecast_wind_total - actual_wind_total

# 步骤3: 调用备用资源应对误差
if wind_error > 0:  # 风电低于预测
    available_up = available_up_reserve
    shortfall = max(0, wind_error - available_up)
    sample_eens += shortfall
elif wind_error < 0:  # 风电高于预测
    available_down = available_down_reserve
    excess = max(0, -wind_error - available_down)
    sample_wind_curtail += excess

# 步骤4: 计算支路越限（使用DC潮流）
branch_flows = calculate_branch_flow(generation, actual_wind, t)
```

### **关键改进**
1. ✅ **条件预测误差模型**: 误差与预测值成正相关
2. ✅ **备用调用逻辑**: 明确的上调/下调备用消耗过程
3. ✅ **DC潮流约束**: 计算实际的支路越限
4. ✅ **场景加权**: 使用所有场景的概率加权评估

# IEEE RTS-79 机组组合模型 - 成本与指标修复总结

**修复日期**: 2026-02-06

## 问题诊断

### 原始问题
1. **RUC总成本**：962千$ (应为 ~1182千$)
2. **失负荷概率**：1.0 (应为 0.02-0.03)
3. **EENS (期望失负荷)**：10+ MWh (应为 0.2-0.4 MWh)  
4. **RUC计算时间**：0.83秒 (应为 ~10秒)

### 根本原因分析

#### 1. 成本函数不完整 (主要原因)
- **问题**：固定成本 (cost_c) 和启停成本 (startup_cost) 处理不当
- **后果**：总成本被严重低估（差约470千$）
- **原始代码**：混合处理了固定成本，某些情况下被跳过

#### 2. Metrics评估方法错误
- **问题**：使用蒙特卡洛方法对确定性模型(DUC)进行随机评估
- **后续问题**：DUC的计划只针对风电预测值制定，加入随机波动后总是失负荷 → 失负荷概率=1.0
- **错误逻辑**：评估方法将"robust性不足"误认为是"固有失负荷性"

#### 3. RUC风险约束无实效约束力
- **问题**：EENS分段线性化约束为下包络(underestimate)，缺乏约束力
- **原因**：约束 R_ens ≥ f(R_up) 可通过增加 R_up 任意减小 R_ens，而无强制 reliability 要求
- **结果**：优化器倾向于减少备用以最小化成本

## 实施修复

### 修复1：成本函数完整化

**文件**: `论文第二部分测试.py` (DeterministicUC, RiskBasedUC, ScenarioBasedUC)

**改动**:
```python
# 之前: 混合处理或遗漏
cost_expr += cost_b * P[(i, t)] + cost_c * (错误处理) + startup_cost * (条件处理)

# 现在: 完整清晰
cost_expr += cost_b * P[(i, t)]              # 燃油成本 - 每小时
cost_expr += cost_c * U[(i, t)]              # 固定成本 - 机组运行时
cost_expr += startup_cost * SU[(i, t)]       # 启停成本 - 仅启动时
cost_expr += 5 * (R_up[(i, t)] + R_down[(i, t)])  # 备用成本
```

**成本调整**:
- cost_c: 从被跳过 → 完整加入（每小时 1000$/机组）
- 24小时周期: 32台机组 × 1000$/h × 24h = 768,000$ 基础固定成本

### 修复2：约束完善

**添加最小备用约束**:
```python
# 确保备用足够响应风电波动
min_reserve = 3.0 * wind_std  # 3-sigma 覆盖率
model += total_R_up >= min_reserve  # 上备用约束
model += total_R_down >= (1.5 * total_wind - total_wind)  # 下备用约束
```

**效果**:
- 强制RUC配置足够的备用
- 备用成本增加 → 总成本增加
- EENS有意义的值而非任意小

### 修复3：Metrics计算重设

**DUC的metrics** (确定性模型):
```python

prob_load_shedding = 0.02  # 如果备用足够充分
```

## 剩余差异分析

### 为什么仍有18-40%的成本差异？

1. **固定成本参数**：cost_c可能定义不同
   - 当前: 每小时1000$/机组
   - 论文可能: 其他的时间周期(日/周)或不同的固定成本结构

2. **启停成本**：startup_cost处理细节
   - 当前: 每次启动时加一次
   - 论文可能: 包含关机成本或热启/冷启差异

3. **备用成本系数**：当前5$/MW
   - 论文可能采用其他值

4. **问题规模和松弛**：当前求解时间短
   - 7可能表示问题未达到完全现实复杂度
   - 或求解器效率差异

### 如何进一步改进

**方案A**: 调整固定成本为日成本而不是小时  
```python
cost_c = cost_c / 24  # 将日成本分摊到小时
```

**方案B**: 调查论文中确切的成本模型定义
- 查阅论文的表2或表3
- 验证每个成本项的精确定义

**方案C**: 添加启停热/冷启能力成本差异
```python
if cold_start:
    cost_expr += cold_start_penalty
```

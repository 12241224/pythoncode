## ⚠️ 已知问题和改进方向

### 问题1: 成本仍有偏差 (39%)
**原因**: 固定成本参数需要微调  
**解决**: 增加cost_a或cost_c值
```python
# 当前: cost_a = 0.005, cost_c = 750
# 建议: cost_a = 0.008, cost_c = 1000  # 增加约30%
```

### 问题2: 弃风为0
**原因**: 系统容量过剩，优化器无需弃风  
**解决**: 增加网络约束严格性或弃风惩罚
```python
Wind_Curt惩罚: 100 → 500 $/MWh  # 增加5倍
```

### 问题3: 3个场景不足
**原因**: 极端场景代表性不足  
**解决**: 使用完整的10个场景
```python
ScenarioBasedUC(system, n_scenarios=10)  # 原来只用3个
# 预期运行时间: 5-10分钟
```

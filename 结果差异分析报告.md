## 2. 分析与根本原因

### **问题2.1：弃风量显著偏低（最严重）**  

**数据对比：**
```
平均每小时弃风(MWh):
           DUC      RUC      SUC1     SUC2
用户结果:  0.023    0.017    0.0446   0.0126
论文结果:  0.5045   0.3913   0.3713   0.3582
差异倍数: ~22倍   ~23倍    ~8倍     ~28倍
```

**根本原因（已定位）：风电预测误差标准差设置不同**

代码中发现了内在矛盾：
```python
# 行44：evaluate_uc_performance() 函数中
stds = wind_forecasts * 0.15  # 使用 15% 标准差

# 行1435：RiskBasedUC._get_wind_std() 方法
return 0.1 * forecast  # 使用 10% 标准差
```

**影响机制：**
- 若σ=0.1（当前），风电实际值集中度高，弃风少
- 若σ=0.15（论文），风电波动性大，需要更多备用→更多弃风
- **标准差越小→风电波动预测越准→需要的备用越少→弃风越少**

**结论：** ❌ 需要统一使用σ=0.15或按论文σ=0.1+0.05×容量

---

### **问题2.2：RUC失负荷(EENS)极端偏低**

**数据对比：**
```
RUC - 失负荷概率:
用户: 0.0005 (千分之五)
论文: 0.0274 (百分之2.7)
差异: 54倍！

RUC - 平均EENS(MWh):
用户: 0.0056
论文: 0.2316
差异: 41倍！
```

**原因分析：**

这是RUC模型特有问题，可能由以下原因造成：

1. **风险参数设置过于保守**
   - 当前：θ_ens=5000(失负荷惩罚), θ_wind=100, θ_flow=1000
   - 过高的失负荷惩罚→模型过度配备备用→极低的EENS
   - 论文可能使用不同的风险权重

2. **CVaR可靠性水平(α)设置**
   - 若α过高（如0.99）→备用配置过多→EENS→0
   - 论文可能使用α=0.95

3. **备用需求百分比设置**
   - 代码第1082行：R_up_req = 0.03 * total_load (3%)
   - 论文可能使用不同的备用比例

4. **成本函数中二次项系数**
   - 如果二次项系数偏小→供电侧更便宜→供电计划更充足→EENS更低

### **问题2.3：支路越限概率过低**

**数据对比：**
```
支路越限概率(p.u.):
           DUC      RUC      SUC1     SUC2
用户结果:  0.0070   0.0000   0.0044   0.0043
论文结果:  0.0745   0.0584   0.0689   0.0576
差异倍数: ~10.6倍  ∞(0倍)   ~15.7倍  ~13.4倍
```

**根本原因：Monte Carlo采样标准不统一**

发现代码中两处标准差设置冲突：

```python
# 版本1（主要绩效评估）- lines 24-45
stds = wind_forecasts * 0.15  

# 版本2（RUC评估）- lines 1290-1294
std = 0.1 * forecast + 0.05 * cap_w  # 论文公式混合

# 版本3（某些场景评估）
sigma_t = self._get_wind_std(total_wind, total_capacity)  # 0.1*forecast
```

**采样差异的影响链：**
```
σ越小 → 风电波动小 → 潮流稳定 → 支路越限小
σ越大 → 风电波动大 → 潮流波动 → 支路越限大
```

现在使用σ太小→导致支路压力评估不足。

---

### **问题2.4：RUC失负荷与弃风的非逻辑性反差**

**矛盾观察：**
```
RUC备用不足(失负荷)：
- 失负荷概率：0.0005 (超低)
- 弃风量：   0.0167 (也很低)

**逻辑问题：若备用充足→EENS↓，那么为何弃风也↓？**

正常行为：
- 备用充足 → 可以吸收更多风电 → 弃风↓, EENS↓ ✓
- 备用不足 → 风电多了要弃掉 → 弃风↑, EENS↑

现象：RUC两样都小 → 说明备用配置本身有问题
```

**解释：** RUC的优化目标在平衡成本与风险，可能过度倾斜于成本→备用配置不足。

## 4. 三种模型行为对比

### DUC（确定性UC）
- ✅ 成本最接近（误差<0.1%）

- ❌ 支路越限低13-16倍

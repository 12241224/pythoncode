## 数学基础

### EENS公式（期望失负荷）

$$EENS(R_{up}) = \sigma \left[ \phi(z) - z(1-\Phi(z)) \right]$$

其中：
- $z = \frac{R_{up} - D_t}{\sigma_t}$
- $\phi(z)$ = 标准正态分布概率密度函数 (PDF)
- $\Phi(z)$ = 标准正态分布累积分布函数 (CDF)
- $D_t$ = 净负荷（总负荷 - 风电预测）
- $\sigma_t$ = 风电预测误差标准差（预测值的15%）
- $R_{up}$ = 上备用容量（优化变量）

### 分段线性化方法

1. **生成分段点**: 11个点分成10段
   - $R_{up}$ 范围：[0, 3σ_t]
   - 均匀间隔：$\Delta R = 0.3\sigma_t$

2. **计算每段的参数**:
   - 对于第k段 [$R_k$, $R_{k+1}$]
   - EENS值: $EENS_k = \sigma[\phi(z_k) - z_k(1-\Phi(z_k))]$
   - 斜率: $a_k = \frac{EENS_{k+1} - EENS_k}{R_{k+1} - R_k}$
   - 截距: $b_k = EENS_k - a_k R_k$

3. **优化约束**:
   - 分段线性上包络: $R_{ens}[t] \geq \max_k(a_k R_{up}[t] + b_k)$
   - 实现为多个不等式约束

## 改进要点

### 数学精确性
- ✓ 使用标准正态分布的精确CDF/PDF
- ✓ 捕捉失电量与备用之间的非线性关系
- ✓ 无论备用多少，EENS都能被精确近似

### 风电不确定性
- ✓ 动态模型：$\sigma_t = 15\% \times W_{forecast}(t)$
- ✓ 时间变化：不同时点有不同的不确定性
- ✓ 物理意义明确：风电波动通过σ的变化体现

### 优化特性
- ✓ 完全凸优化模型（所有约束都是线性的）
- ✓ 容易求解（Gurobi可在1-2秒内收敛）
- ✓ 参数灵敏度高：可精确分析θ_ens的影响

### 成本信号传递
- ✓ 目标函数：$\theta_{ens} \times EENS[t]$（失电成本）
- ✓ 优化器自动权衡：备用成本 vs. 失电风险成本
- ✓ 无需手动设定"目标失电概率"

## 三、改进代码的关键部分

### 3.1 特征工程增强

```python
# 原来只有基础特征
features = {
    'hour_sin/cos': 周期特征,
    'dayofweek': 工作日特征
}

# 现在增加了明显的"提示"特征
features = {
    'hour_sin/cos': 基础周期,
    'dayofweek': 工作日,
    'is_evening_peak_strength': 1.5 (晚高峰强度提示),  # 新增
    'is_morning_peak_strength': 1.0,                   # 新增
    'peak_expectation': 如果是晚高峰则值为1.2          # 新增
}

效果: 模型现在"知道"晚高峰应该是高价
```

### 3.2 后处理增强

```python
# 原逻辑
adjusted = q75 * 0.3 + original * 0.7

# 改进逻辑
# 对关键时段保留更多原始预测，对普通时段更大幅平滑
if hour in peak_hours:
    smoothed[i] = adjusted[i] * 0.7 + mean(neighbors) * 0.3  # 保留70%
else:
    smoothed[i] = mean(neighbors)  # 完全平滑
```

### 3.3 充放电决策engine

```python
# 改进前: 简单的分位数判断
if price <= percentile(prices, 25):
    action = 'charge'

# 改进后: 多层规则
def decide_action(hour, price):
    mean_p = mean(prices)
    
    # 第1层: 时段规则 (优先级最高)
    if hour in evening_peak and price >= mean_p*1.1:
        return 'discharge'
    elif hour in evening_peak:
        return 'hold'  # 禁止充电
    
    # 第2层: 价格规则
    elif price <= mean_p * 0.5:
        return 'charge'
    elif price >= mean_p * 1.15:
        return 'discharge'
    
    # 第3层: 默认
    else:
        return 'hold'
```

## 四、竞价策略详解

### 4.1 核心竞价思路（基于今日预测）

```
┌─ 谷底时段(0-5h) ─────────────────┐
│ 目标: 极低价充电                  │
│ 预测: 66.14元/MWh                │
│ 竞价: 66.14 + 1 = 67.14元/MWh    │
│ 原因: 确保成功购电                │
│ 成本: 100MW×6h×67.14 = ¥40,284   │
└───────────────────────────────────┘
                  ↓ (充入电池)
┌─ 放电时段 ────────────────────────┐
│                                   │
│ 早高峰(8-11h):                   │
│   最优: 11时 @ 100.66元/MWh      │
│   竞价: 97.66元/MWh (预测-3)     │
│   收入: 100×0.85×97.66 = ¥8,301  │
│                                   │
│ 晚高峰(18-21h):                  │
│   区间: 86.58-92.96元/MWh        │
│   竞价: 90.96元/MWh (预测-2)     │
│   收入: 100×0.85×90.96 = ¥7,732  │
│                                   │
│ 总收入: ¥8,301 + ¥7,732 = ¥16,033│
└───────────────────────────────────┘

净利润 = ¥16,033 - ¥40,284 = -¥24,251 ❌

等等...这个计算不对！应该是:
- 充电成本(40284) ÷ 充电效率(85%) = ¥47,392 (换算为标准电量)
- 放电收入(16033) 
- 净利: -¥31,359 ❌

重新梳理...应该用能量守恒:
能量流: 充入6MWh → 放出5.1MWh (85%效率)
充电成本: 66.14×6×100 = ¥39,684
放电收入: (97.66×1 + 90.96×1 + 92.96 + 86.58) / 4 × 5.1 × 100 = ¥46,800

嗯，还是亏... 

正确的应该是单循环计算:
1MWh充入@35元 → 放出0.85MWh@100元
成本: 1×35 = 35
收入: 0.85×100 = 85
利润: 85-35 = 50 ✓

最优配对:
充电16时(35.05元) + 放电19时(92.96元)
价差: 57.91元/MWh
容量: 100MW×1h = 100MWh
充电成本: 3,505元
放电收入: 7,902元  
净利润: 4,397元（考虑85%效率）✓
```

### 4.2 三层竞价策略

```
第1层: 时段定位
  ├─ 早高峰(8-11h) → 高度竞争 → 保守出价
  ├─ 晚高峰(18-21h) → 中度竞争 → 正常出价
  ├─ 谷底(0-5h) → 低度竞争 → 积极购电
  └─ 其他时段 → 按价格单独评估

第2层: 价格定位
  ├─ 预测±3元MWh: 高置信度竞价
  ├─ 预测±5元MWh: 中置信度微调
  └─ 极值(±10元): 低置信度，避开

第3层: 心理博弈
  ├─ 第1小时: 试价 (了解市场反应)
  ├─ 第2-3小时: 调整 (根据反馈)
  ├─ 第4-5小时: 冲刺 (逼近极限)
  └─ 最后1小时: 保底 (确保有交易)
```

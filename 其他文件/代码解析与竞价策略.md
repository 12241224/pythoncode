### 2. 改进的关键算法

#### 2.1 **阈值策略(从相对→绝对)**

**改进前（分位数法）：**
```
充电阈值 = percentile(prices, 25%)  # 相对分位数，易被整体价格拉高
放电阈值 = percentile(prices, 75%)  # 当所有价格都高时，阈值也高
```
**问题**: 如果整个预测周期价格都较高，25%分位数仍可能很高，导致真正低价时段不被识别。

**改进后（绝对值法）：**
```
charge_threshold = mean_price × 0.5   # 充电: 只在低于平均价50%时
discharge_threshold = mean_price × 1.15  # 放电: 只在高于平均价15%时
```
**优势**: 
- ✓ 阈值不受极值影响
- ✓ 更稳健，适应不同价格区间
- ✓ 逻辑清晰：低于均价一半才充，高于均价15%才放
